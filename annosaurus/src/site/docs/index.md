# annosaurus

![MBARI logo](assets/images/logo-mbari-3b.png)

## Overview

Annosaurus is a REST API for creating and managing video and image annotations. It is a core service of MBARI's [Video Annotation and Reference System](https://github.com/mbari-org/m3-quickstart), providing a language-agnostic interface so annotations can be created and retrieved from any programming environment.

The service requires only a database (PostgreSQL or SQL Server) and exposes a Swagger UI at `/docs` once running.

Source code and issue tracker: <https://github.com/mbari-org/annosaurus>
